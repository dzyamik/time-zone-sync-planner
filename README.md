# Timezone syncronization

## Description

A tool to find overlapping times for remote team meetings. Using Vite + Vue + JS

## Requirements

`Node.js v22.14.0`

## Commands

### Init project

```bash
npm install
```

### Run project

```bash
npm run dev
```

### Make a build

```bash
npm run build
```

### Preview a build

```bash
npm run preview
```

### Deploy build on github pages

```bash
npm run deploy
```

### Work with packages

```bash
rm -rf node_modules package-lock.json
npm cache clean --force
```
